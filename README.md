# PDeISC - Repositorio principal

Este repositorio funciona como índice principal para guardar y ordenar los trabajos de **PDeISC / PDelSC**.

La idea es tener un lugar central desde donde se pueda acceder a todos los trabajos importantes de la materia, desde el primer repositorio de PDeISC hasta **PDelSC Formularios JS 2**.

## Objetivo

- Tener todos los trabajos organizados en un solo lugar.
- Mantener los nombres claros y ordenados.
- Acceder rápido a cada entrega.
- Poder seguir agregando nuevos trabajos de la materia sin desordenar el GitHub.

## Cómo agregar un nuevo trabajo

1. Crear una carpeta dentro de `trabajos/`.
2. Usar un número al inicio para mantener el orden.
3. Agregar un `README.md` dentro de la carpeta explicando de qué trata.
4. Subir los archivos del trabajo o colocar el enlace al repositorio correspondiente.

Ejemplo:

```text
trabajos/06-Nuevo-Trabajo/
└── README.md
```
